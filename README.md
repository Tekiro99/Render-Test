# Popscoin Realm

Минималистичная realtime 2D multiplayer-игра для деплоя на Render.

## Что есть

- Лобби с кодом и ссылкой-приглашением
- Общая большая карта
- Минималистичный редактор персонажа
- Перемещение на `WASD`
- Взаимодействие с объектами на `E`
- Майнинг залежи в центре карты с получением попскойнов
- Город с магазином: кирка повышает доход, меч повышает урон
- Враги на карте и атака на `Space`

## Запуск локально

```bash
npm install
npm start
```

Открой `http://localhost:3000`.

## Deploy на Render

1. Залей репозиторий в GitHub.
2. Создай Web Service в Render.
3. Render сам подхватит `render.yaml`, либо укажи:
   - Build Command: `npm install`
   - Start Command: `npm start`

## Ограничения текущего прототипа

- Состояние хранится в памяти процесса
- При рестарте сервера лобби и прогресс пропадают
- Пригласительная ссылка сейчас отображается как шаблонный URL с кодом лобби
